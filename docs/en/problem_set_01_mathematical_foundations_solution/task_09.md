# Problem 9 — Harmonic oscillator

The equation is

d²x/dt² + ω² x = 0

---

# 1. General solution

The general solution of the harmonic oscillator is

x(t) = A cos(ωt) + B sin(ωt)

where A and B are constants determined by initial conditions.

---

# 2. Velocity

Velocity is the derivative of position:

v(t) = dx/dt

v(t) = −Aω sin(ωt) + Bω cos(ωt)

---

# 3. Acceleration

Acceleration is the second derivative:

a(t) = d²x/dt²

a(t) = −Aω² cos(ωt) − Bω² sin(ωt)

This can also be written as

a(t) = −ω² x(t)

---

# Physical interpretation

The system performs **harmonic oscillations**.

The acceleration is always directed toward the equilibrium position and is proportional to the displacement.
