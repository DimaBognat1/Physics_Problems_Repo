# Problem 4 — Geometry of parametric curves

We investigate several parametric curves.

---

# A) x(t) = R cos(t),  y(t) = R sin(t)

## 1. Eliminate the parameter

x² + y² = R²

This is the equation of a **circle** with radius R.

---

## 2. Type of curve

Circle.

---

## 3. Velocity

v(t) = (dx/dt, dy/dt)

dx/dt = -R sin(t)  
dy/dt = R cos(t)

v(t) = (-R sin(t), R cos(t))

---

## 4. Acceleration

a(t) = (d²x/dt², d²y/dt²)

d²x/dt² = -R cos(t)  
d²y/dt² = -R sin(t)

a(t) = (-R cos(t), -R sin(t))

---

## 5. Magnitude

|v| = √(R²sin²t + R²cos²t)

|v| = R

Velocity magnitude is **constant**.

|a| = R

Acceleration magnitude is **constant**.

---

# B) x(t) = a cos(t),  y(t) = b sin(t)

## 1. Eliminate the parameter

x²/a² + y²/b² = 1

---

## 2. Type of curve

Ellipse.

---

## 3. Velocity

v(t) = (-a sin(t), b cos(t))

---

## 4. Acceleration

a(t) = (-a cos(t), -b sin(t))

---

## 5. Magnitude

|v| = √(a²sin²t + b²cos²t)

Velocity magnitude **is not constant**.

---

# C) x(t) = t,  y(t) = t²

## 1. Eliminate parameter

t = x

y = x²

---

## 2. Type of curve

Parabola.

---

## 3. Velocity

v(t) = (1, 2t)

---

## 4. Acceleration

a(t) = (0, 2)

---

## 5. Magnitude

|v| = √(1 + 4t²)

Velocity magnitude **is not constant**.

|a| = 2

Acceleration magnitude **is constant**.

---

# D) x(t) = cosh(t),  y(t) = sinh(t)

## 1. Eliminate parameter

x² - y² = 1

---

## 2. Type of curve

Hyperbola.

---

## 3. Velocity

v(t) = (sinh(t), cosh(t))

---

## 4. Acceleration

a(t) = (cosh(t), sinh(t))

---

## 5. Magnitude

|v| = √(sinh²t + cosh²t)

Velocity magnitude **is not constant**.
